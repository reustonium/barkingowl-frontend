URL Dispatch

This page allows you to upload a CSV file with a list of URLs to be dispatched to the barkingowl scraper system. The URLs will be sent to the dispatcher, and then dispatched to all of the scrapers that are listening on the message bus. Follow the following steps to upload a CSV, ensure it is valid, and then submit the list of URLs to the dispatcher.
1. Select a CSV File
2. Verify the CSV File
2b. Check JSON
3. Submit URLs for Dispatch
